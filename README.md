# 🕵️ Дело о пропавшем артефакте Короля

Детективный квест для Атасии. Один файл `index.html` — открывается в браузере, работает на телефоне и компьютере, сохраняет прогресс.

## 📁 Структура
```
mj-quest/
├── index.html      ← весь квест здесь
├── img/            ← (необязательно) свои картинки img1.png … img5.png
└── README.md       ← эта инструкция
```
Картинки НЕ обязательны — без них квест выглядит красиво за счёт CSS/SVG. Хочешь добавить — клади в `img/` с именами `img1.png`–`img5.png`.

## 🔑 Ответы (шпаргалка для тебя)
1. Монстр в ночи → **THRILLER** (или «триллер»)
2. Певчая улика → **ПТИЦА**
3. Шаг назад → **WALK**
4. Тайна специй → **0608**
5. Послание Короля → **NEVERLAND**
6. Портал в раме → **ВЫШЕ** (или «вверх», «наверх»)
7. Корона на вершине → **МАЙКЛ ДЖЕКСОН**

Регистр, пробелы и знаки препинания не важны.

## 🚀 Как опубликовать

### Vercel (проще всего)
1. Зайди на vercel.com → New Project.
2. Перетащи папку `mj-quest` (или загрузи через Git).
3. Framework: **Other**. Готово — получишь ссылку.

### GitHub Pages
1. Создай репозиторий, залей файлы.
2. Settings → Pages → Source: ветка `main`, папка `/root`.
3. Через минуту сайт будет по адресу `https://<логин>.github.io/<репозиторий>/`.

## 🖼️ Картинки (если захочешь сгенерировать)
Промпты — в чате. Имена файлов: `img1.png` … `img5.png`, папка `img/`.
Код подхватит их автоматически, если решишь использовать.

## ⚙️ Прогресс
Сохраняется в браузере (localStorage). Закрыла вкладку — откроет с того же места.
Кнопка «Сбросить расследование» внизу — на случай, если захочешь начать заново или протестировать.
